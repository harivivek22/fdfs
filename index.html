Firebase Setup — One Time Only

Step 1. Go to console.firebase.google.com
Step 2. Click Add project → name it anything → Continue
Step 3. In your project → click Web icon (</>) → register app
Step 4. Copy the firebaseConfig object shown
Step 5. In the left menu → Build → Realtime Database
Step 6. Click Create Database → choose Test mode → Enable
Step 7. Open index.html in a text editor
Step 8. Find // PASTE YOUR FIREBASE CONFIG near the top
Step 9. Replace the placeholder values with your copied config
Step 10. Save and re-upload to GitHub — done! 🎉
